# NANDA Town Project

Agent-to-agent services and skills built for [Nanda Town](https://nandatown.projectnanda.org) by Project NANDA.

## Services

### contract-agent
Generates, prices, and notarizes A2A service contracts using a token-premium pricing model.
Integrates with the Town Notary for countersignature.

- **API:** `contract-agent/api/` — FastAPI, deployed on Railway
- **Skill:** `contract-agent/skills/hackathon-contract-agent/SKILL.md`
- **Deploy guide:** `contract-agent/docs/DEPLOY.md`

## Registry

Skills are submitted to the Nanda Town registry at https://nandatown.projectnanda.org/skills
